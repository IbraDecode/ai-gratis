SIbra AI API

Multi-engine AI API. Chat dengan Gemini, Standard, atau GPT. by Ibra Decode.

Base URLhttps://ai-gratis-by-ibra-decode.vercel.app
Version2.2.0
AuthorIbra Decode

SIbra AI adalah API yang menggabungkan beberapa engine AI dalam satu endpoint. Pilih model sesuai kebutuhan, dan dapatkan response dalam format JSON atau streaming.

Models

IDNameEngineStatus
geminiGeminiGoogle Geminiactive
unlimited-stdStandardUnlimitedAIactive
gptGPT-5.4 MiniGPTactive

Authentication

Tidak perlu authentication. API ini gratis dan terbuka untuk umum.

Chat

Kirim prompt ke AI dan dapatkan response.

POST /api/chat

Request Body

{
  "prompt": "halo siapa kamu?",
  "model": "gemini",
  "stream": false,
  "session": null
}
FieldTypeDefaultDescription
promptstring—wajib, maks 10.000 karakter
modelstringgeminiID model
streambooleanfalsestreaming SSE
sessionstringnullsession ID untuk memori percakapan

Response

{
  "status": true,
  "model": "gemini",
  "prompt": "halo siapa kamu?",
  "response": "Halo! Saya Gemini...",
  "session": "uuid"
}

cURL

curl -X POST https://ai-gratis-by-ibra-decode.vercel.app/api/chat \
  -H "Content-Type: application/json" \
  -d '{"prompt":"cerita pendek","model":"gemini"}'

Streaming

Mendapatkan response secara real-time via SSE (Server-Sent Events).

POST /api/chat dengan parameter stream: true 
curl -N -X POST https://ai-gratis-by-ibra-decode.vercel.app/api/chat \
  -H "Content-Type: application/json" \
  -d '{"prompt":"cerita pendek","model":"gemini","stream":true}'

Streaming response format:

data: {"delta":"Halo"}
data: {"delta":"! Saya"}
data: {"delta":" Gemini"}
data: {"done":true,"session":"uuid"}
